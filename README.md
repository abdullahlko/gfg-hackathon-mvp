## Project Overview

**NiveshSathi** is a beginner-friendly investment education and guidance web application designed to make **investing simple, accessible, and engaging for everyone**.

Many first-time investors face challenges such as:
- Lack of financial literacy
- Confusion between investment options like FDs and Mutual Funds
- Difficulty choosing investments based on goals and risk

**NiveshSathi** addresses these problems by:
- Explaining financial concepts in **simple, easy-to-understand language**
- Providing **visual learning through videos and images**
- Guiding users toward **informed and confident investment decisions**

This project is built in alignment with **Problem Statement 13: Investing for Everyone**.
